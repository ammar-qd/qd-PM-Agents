# Qordata Market Research Agent

Shared GitHub Copilot / Claude prompt workflows for Product Managers and GTM teams.

## What this repo provides

This repo contains reusable prompt files and instructions for structured market research in VS Code.

## Included workflow

- Market research
  - competitive analysis
  - market sizing
  - trend analysis
  - due diligence on market segments

## How to use in VS Code

### 1. Open this repository in VS Code
Clone the repo and open it as your workspace.

### 2. Open GitHub Copilot Chat
Use GitHub Copilot Chat inside VS Code.

### 3. Attach or invoke the prompt file
Use the market research prompt file from:
`.github/prompts/market-research.prompt.md`

### 4. Ask for your research task
Example:
`Research the market for Aggregate Spend tools and create a competitive analysis`

### 5. Add more context when useful
Examples:
- geography
- industry
- customer segment
- company size
- desired output format
- depth of analysis

## Example prompts

- Research the market for Aggregate Spend tools and create a competitive analysis
- Estimate the market size for AI HCP Verification tools in US
- Analyze trends in the Expense Monitoring software market for roadmap planning
- Do due diligence on customer data platforms for HCP Engage B2B SaaS

## Recommended prompt structure

Use this template:

- Topic: [market/category]
- Goal: [competitive analysis / market sizing / trend analysis / due diligence]
- Geography: [region]
- Customer segment: [SMB / mid-market / enterprise / industry]
- Audience: [PM / GTM]
- Output format: [memo / markdown report / table / brief / slide outline]
- Must include: [competitors / trends / pricing / recommendations / citations]

## Notes

- Live web research depends on the active Copilot/Claude setup and enabled capabilities.
- Always validate important claims before sharing externally.
- Prefer recent sources and cite them.
